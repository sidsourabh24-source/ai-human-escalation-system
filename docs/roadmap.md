# Project Roadmap

## Milestone 1 - Foundation (Week 1)
- Initialize monorepo with frontend and backend
- Add branch strategy and PR rules
- Set up Kanban workflow
- Define issue templates

## Milestone 2 - Backend Core (Week 2)
- Build Express API + Socket.io server
- Add DB schema and migration SQL
- Add Claude integration layer
- Add JWT auth skeleton

## Milestone 3 - AI to Human Handoff (Week 3)
- Build customer chat widget
- Detect escalation signals
- Pause AI and begin human handoff
- Preserve transcript context

## Milestone 4 - Agent Operations (Week 4)
- Agent queue and claim flow
- Agent messaging in dashboard
- Escalation and lead email alerts
- Audit trails and logs

## Milestone 5 - Lead Pipeline + Final Polish (Week 5)
- Lead extraction from chat
- Mocked HubSpot sync
- Reliability hardening
- Final documentation and demo runbook
